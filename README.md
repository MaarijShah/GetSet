**GetSet – Q&A Social Networking Website**
**Overview**
GetSet is a web-based Question & Answer social networking platform where users can ask questions, provide answers, and interact through comments. Users can also browse questions by categories and find relevant discussions easily.
The platform is designed to support knowledge sharing and improve communication between users with similar interests.
**Purpose**
The purpose of this project is to develop a simple and interactive system where users can ask questions, receive answers, and engage in discussions in an organized online environment.
It aims to improve knowledge sharing and create a structured Q&A community.
** Features**
User registration and login system
Ask and answer questions
Comment on posts for interaction
Category-based question browsing
Search functionality for easy access
Admin panel for managing users and content
Role-based access (Admin / User)
Simple and user-friendly interface

**How to Run the Project Locally**
1. Install Requirements
Make sure you have:
XAMPP installed (Apache + MySQL / SQL Server setup)
A web browser
2. Setup Project Folder
Place the project folder inside:
xampp/htdocs/
Example:
xampp/htdocs/getset_project
3. Start Server
Open XAMPP Control Panel
Start Apache
Start MySQL (if required)
4. Database Setup
Open phpMyAdmin or SQL Server tool
Create a database for the project
Import the provided database file (if available in project)
5. Run Project
Open browser and go to:
http://localhost/getset_project/
